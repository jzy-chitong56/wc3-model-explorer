[SIZE=5][B]Overview[/B][/SIZE]

WC3 Model Explorer is a standalone desktop application for browsing and previewing Warcraft III model files (MDX/MDL, no HD). Point it at a folder — or open a map archive — and instantly get a 3D thumbnail grid of every model inside. Click any thumbnail to open a full 3D viewer with animation playback, texture inspection, UV maps, bone trees, and more.

[URL='https://www.hiveworkshop.com/media/wc3-model-explorer.144731/'][B]Video demonstration here[/B][/URL]

[URL='https://github.com/Jaccouille/wc3-model-explorer'][B]GitHub Repository[/B][/URL]

[SIZE=4][B]Supported Formats[/B][/SIZE]
[LIST]
[*][B]Model files:[/B] Should handle MDX 800 to 1200. Doesn't support Reforged HD models.
[*][B]Map archives:[/B] .w3x / .w3m — Warcraft III Reforged map format. Older (pre-Reforged) maps using MPQ format are also supported.
[/LIST]

[SIZE=5][B]Features[/B][/SIZE]

[SIZE=4][B]Model Browser[/B][/SIZE]
[LIST]
[*]Recursive scanning of directories and map archives ([B].w3x[/B], [B].w3m[/B])
[*]3D thumbnail grid with configurable size (128–352px) and render quality
[*]Real-time search by filename or internal model name
[*]Sort by name, file size, or parent folder
[*]Favorites and recent models tracking
[*]Advanced filters: animation name, texture path, polygon range, file size range, portrait filter
[*]Team color selector for thumbnails
[*]Right-click context menu: copy path/file, open file location, open in external program, delete
[*]Drag & drop models to external applications (copies the file)
[/LIST]

[SIZE=4][B]3D Model Viewer[/B][/SIZE]
[LIST]
[*]Orbit, pan, and zoom camera with mouse controls
[*]7 shading modes: Solid, Textured, Lit, Normals, Geoset Colors, Bone Count heat map, Wireframe
[*]Animation playback with sequence selector, play/pause, loop, speed control (0.1x–3x)
[*]Model camera view toggle — use the camera defined in the model file
[*]Overlay toggles: extents, bones, helpers, attachments, ribbon emitters, particle emitters, collision shapes, node names, grid
[*]Screenshot export to PNG
[*]On-screen stats HUD (geosets, vertices, triangles, bones, sequences)
[/LIST]

[SIZE=4][B]Model Inspection Tabs[/B][/SIZE]
[LIST]
[*][B]Animation[/B] — sequence list with duration, scrubber, speed control, loop, recenter
[*][B]Info[/B] — file name, model name, path, size, vertex/polygon/bone/geoset/sequence/texture counts, bounding radius
[*][B]Textures[/B] — texture list with source indicator (CASC/MPQ/disk), hover preview, zoomable popup
[*][B]Materials[/B] — material layers with filter modes and geoset associations
[*][B]Geosets[/B] — per-geoset vertex count, material, filter mode — click to highlight in the 3D view
[*][B]UV Map[/B] — per-geoset/layer UV visualization with wireframe and alpha toggles
[*][B]Nodes[/B] — bone tree hierarchy — click to highlight in the 3D view
[/LIST]

[SIZE=4][B]Data Sources & Texture Resolution[/B][/SIZE]

Models reference textures from game archives. WC3 Model Explorer can load textures from:
[LIST]
[*][B]CASC[/B] — Warcraft III Reforged installations (auto-detected)
[*][B]MPQ[/B] — classic Warcraft III archives (multiple archives supported)
[*][B]Disk[/B] — textures alongside model files
[/LIST]
Supported texture formats: [B]BLP[/B], [B]DDS[/B], [B]TGA[/B], [B]PNG[/B], [B]JPEG[/B]

[SIZE=5][B]Getting Started[/B][/SIZE]

[SIZE=4][B]1. Browse models[/B][/SIZE]
Click [B]Browse[/B] and select a folder containing [ICODE].mdx[/ICODE] / [ICODE].mdl[/ICODE] files. The application scans recursively and displays animated 3D thumbnails. You can also open map archives ([ICODE].w3x[/ICODE], [ICODE].w3m[/ICODE]) directly.

[SIZE=4][B]2. Configure data sources (optional)[/B][/SIZE]
To display game textures on models, open [B]Settings[/B] and configure a data source:

[B]CASC (Reforged):[/B] Click Browse and select your Warcraft III install folder (the one containing [ICODE]_retail_[/ICODE] or [ICODE]Data[/ICODE]).
Typical path: [ICODE]C:\Program Files (x86)\Warcraft III[/ICODE]

[B]MPQ (Classic):[/B] Click Add and select your [ICODE].mpq[/ICODE] archive files ([ICODE]war3.mpq[/ICODE], [ICODE]war3x.mpq[/ICODE], [ICODE]war3patch.mpq[/ICODE]). Multiple archives can be added.

[SIZE=4][B]3. Explore a model[/B][/SIZE]
Click any thumbnail to open the 3D viewer. Use the tabs on the right to inspect animations, textures, materials, geosets, UV maps, and the bone hierarchy.
